- Запустить сервер. 
- Сменить флаг с "OFF" на "ON". Высветится уведомление о том, что сервер запущен. 
- При нажатии на кнопку "Свернуть сервер в трей" сервер свернется в трей, снова его можно будет открыть нажатием ЛКМ. 
- Чтобы изменить настройки, необходимо вписать хост и порт. После этого необходимо нажать кнопку "Изменить настройки". После изменения настроек необходимо перезапустить сервер.

- Запустить клиент.
- Ввести ник обязательно, порт можно оставить пустым, и нажать на кнопку "ОК".
- Высветится уведомление об успешности подключения. 
- Если настройки сервера были изменены, а сам сервер перезапущен, то для дальнейшей работы клиенту необходимо будет при запуске вписывать еще и порт.
